
# 💻 Exercícios de Programação Orientada a Objetos (POO)

😘**Acesse aqui o projeto:**
[Ex de POO](https://baccarinha.github.io/exerciciopoorafaelcavallari/)

🎯 Objetivo do Projeto

Desenvolver um ambiente digital para apresentação de exercícios de Programação Orientada a Objetos, permitindo a prática e o entendimento dos conceitos fundamentais dessa abordagem de programação.

|🧩 Funcionalidades|
|----|
📚 Exibição de exercícios de Programação Orientada a Objetos
🧠 Demonstração de conceitos fundamentais da POO
🖥️ Interface simples e organizada
📱 Layout acessível via navegador
📖 Organização lógica dos conteúdos
🎓 Projeto voltado ao aprendizado acadêmico
🛠️ Tecnologias Utilizadas


|O projeto foi desenvolvido utilizando:|
|----|
HTML5 — Estruturação das páginas
CSS3 — Estilização visual
JavaScript — Interatividade e navegação
GitHub Pages — Hospedagem do projeto
Inteligência Artificial (IA) — Apoio na criação das ideias e estrutura
