Cálculo de Média (5 Notas)

Código usando FOR (Python)

soma = 0

for i in range(1, 6):
    nota = float(input(f"Digite a {i}ª nota: "))
    soma += nota

media = soma / 5

print(f"Média das notas: {media}")

Código usando WHILE (Python)

soma = 0
contador = 1

while contador <= 5:
    nota = float(input(f"Digite a {contador}ª nota: "))
    soma += nota
    contador += 1

media = soma / 5

print(f"Média das notas: {media}")
